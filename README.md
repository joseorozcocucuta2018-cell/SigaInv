# SIGAINV
### Sistema de Gestión de Inventarios

**Simplificando la administración de tu negocio**

SIGAINV es un sistema integral diseñado para administrar inventarios, ventas, compras y movimientos de productos. Permite controlar el stock, gestionar documentos comerciales y realizar seguimiento financiero de una empresa.

---

# Características

## 📦 Gestión de Inventario
Control completo de productos, manejo de stock por bodega, administración de lotes con fechas de vencimiento y números de serie.

## 🛒 Ventas y Comercial
Gestión de cotizaciones, remisiones y facturas con control de estados y seguimiento de pagos.

## 🏬 Compras
Registro de órdenes a proveedores, control de entregas y seguimiento de pagos pendientes.

## 🔄 Transformaciones
Creación de fórmulas para manufactura, empaques o combos con generación automática de movimientos de inventario.

## 🚚 Traslados
Traslado de productos entre bodegas con confirmación y seguimiento.

## 🏦 Bancos y Cajas
Control de movimientos bancarios, caja diaria y transferencia de fondos.
